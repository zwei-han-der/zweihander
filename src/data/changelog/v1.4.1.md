Implementada a troca dinâmica de favicon com base no tema carregado.

## Adicionado

- Campo `favicon` em todos os temas de `src/data/themes.js`
- Mapeamento de ícones versionados (`/favicons/themeX.svg?v=1`) para reduzir risco de cache agressivo
- Atualização dinâmica de `link[rel='icon']` no carregamento do site

## Modificado

- Aplicação de tema em `src/App.jsx` agora limita CSS variables apenas às chaves de cor (`color*`)
- Fluxo de inicialização do tema passou a incluir fallback para favicon padrão (`/favicon.svg`)
- Estrutura textual dos arquivos de changelog padronizada para manter consistência de leitura

## Corrigido

- Removida tentativa de aplicar `favicon` como CSS custom property (`--favicon`)
