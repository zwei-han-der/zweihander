Primeira versão estável do projeto, consolidando as principais páginas e componentes da base.

## Adicionado

- Suporte completo a Markdown no `MarkdownRenderer` com `remark-gfm` e `rehype-highlight`
- Renderização de `checkbox`, `table`, `strikethrough`, `img` e blocos de código com syntax highlight
- Modal de changelog com renderização de conteúdo em Markdown
- Listagem dinâmica de vídeos com dados em `src/data/videos.js`
- Modal de vídeo com reprodução e metadados do item selecionado
- Player de música com controles de faixa, playlist personalizada e visual integrado ao tema
- Temas de fundo dinâmicos via CSS custom properties (`--color01..--color04`)
- Estrutura de layouts `Default` e `Standalone`

## Modificado

- Revisão de estilo das tags Markdown (`h1` a `h3`, `p`, `ul`, `ol`, `li`, `a`, `blockquote`, `code`, `pre`, `table`, `checkbox`)
- Organização do modal como componente próprio com render via `React.createPortal`
- Ajustes visuais para manter linguagem de glassmorphism em páginas e componentes

## Corrigido

- Correções de comportamento para fechamento de modal via botão, `Esc` e clique fora
- Correções de renderização para blocos Markdown customizados

## Detalhes tecnicos

- Bundler: `vite`
- Gerenciador de pacotes: `bun`
- Roteamento: `react-router-dom`
- Renderização Markdown: `react-markdown`, `remark-gfm`, `rehype-highlight`
