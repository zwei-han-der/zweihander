Aba de vídeos pronta, feita para armazenar, principalmente, minhas jogadas no basquete.

## Funcionalidades

- Layout utilizado: Standalone
- Grid responsivo: 3 colunas -> 2 colunas (`max-width: 900px`) -> 1 coluna (`max-width: 600px`)
- Cards com thumbnail, título e data
- Clique no card abre Modal com player de vídeo com os seguintes atributos: `autoPlay`, `loop` e `muted`
