Melhorias de performance com route-based code splitting e lazy loading, além de novas rotas para changelog e blog.

## Adicionado

- Code splitting com `lazy()` e `Suspense` em `App.jsx`
- Lazy loading para as abas `Home`, `Changelog`, `Blog`, `Videos`, `Bookmarks` e `NotFound`
- Lazy loading para `MarkdownRenderer`
- Rotas filhas para acesso direto em `/changelog/:logId` e `/blog/:postId`
- Aba [Blog](/blog) com sistema de posts e cobertura
- Cache em memória (`Map`) para conteúdo de posts e logs
- Suporte a deep linking por parâmetros de URL
