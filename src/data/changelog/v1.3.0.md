Adicionado parser HTML ao MarkdownRenderer, permitindo renderização de HTML bruto dentro do conteúdo Markdown.

## Adicionado

- Suporte a HTML embutido em Markdown com `rehype-raw`
- Renderização de `<hr>` com classe dedicada (`md-hr`)
- Estilo para separadores horizontais (`.md-hr`) com borda sutil e espaçamento vertical

## Modificado

- Nome do projeto atualizado de `zwei` para `zweihander`
- Dependências e ferramentas de build/lint atualizadas
