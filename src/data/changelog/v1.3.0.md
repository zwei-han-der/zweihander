Adicionado parser HTML ao MarkdownRenderer, com renderização de HTML bruto dentro do conteúdo Markdown e estilização para separadores horizontais.

## Adicionado

- Suporte a HTML embutido em Markdown com `rehype-raw`
- Renderização de `<hr>` com classe dedicada (`md-hr`)
- Adicionado estilo para separadores horizontais no Markdown (`.md-hr`), com borda sutil e espaçamento vertical

## Modificado

- Nome do projeto atualizado de `zwei` para `zweihander`
- Versionamento do pacote ajustado para `1.3.0`
- Dependências e ferramentas de build/lint atualizadas
