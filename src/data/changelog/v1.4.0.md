Introduzido o sistema de preview de links com geração de metadados no build e integração no Markdown/Bookmarks.

## Adicionado

- Verificação de resolução das imagens para evitar thumbs de baixa qualidade ou cortadas
- Captura de metadados de resolução no pipeline de geração de previews
- Gerador de metadados no prebuild
- Componente `LinkPreview` com loader de metadados
- Integração de `LinkPreview` em links do Markdown e da aba Bookmarks

## Modificado

- Lógica de renderização do LinkPreview atomizada para facilitar manutenção
- Estrutura do script de metadados atualizada para incluir resolução da imagem

## Corrigido

- Fallback automático para URL bruta quando a descrição é inválida ou pobre
