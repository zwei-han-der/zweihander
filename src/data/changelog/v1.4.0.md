Introdução do sistema de preview de links, cobrindo geração de metadados no build, integração no conteúdo Markdown/Bookmarks e melhorias de qualidade visual e textual.

## Adicionado

- Verificação de resolução das imagens de preview para evitar thumbs de baixa qualidade
- Captura e uso de metadados de resolução no pipeline de geração de previews
- Gerador de metadados no prebuild para alimentar previews de links
- Componente `LinkPreview` com loader de metadados
- Integração de `LinkPreview` nos links renderizados no Markdown e na aba Bookmarks

## Modificado

- Lógica de renderização do LinkPreview atomizada para separar lógica e facilitar manutenção
- Estrutura do script de metadados atualizada para incluir dados de resolução de imagem
- Novo visual glass para o tooltip de preview, com apresentação horizontal da informação

## Corrigido

- Quando a descrição for inválida/pobre, fallback automático para URL bruta em vez de texto quebrado
